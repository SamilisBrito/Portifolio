## 📋 Sobre o Projeto

Portfólio profissional de **Samilis Brito**, desenvolvedora front-end especializada em interfaces modernas, performáticas e acessíveis. O site foi construído com foco em **performance**, **semântica HTML** e **acessibilidade (WCAG)**, utilizando uma stack leve e sem frameworks pesados.

## 🛠️ Tecnologias

- **HTML5** — Estrutura semântica com ARIA labels e navegação acessível
- **CSS3** — Vanilla CSS com custom properties, grid layout, animações e design responsivo
- **TypeScript** — Lógica de interação (menu mobile, smooth scroll, header dinâmico)
- **Google Fonts** — DM Sans + JetBrains Mono

## 🎨 Destaques de Design & Acessibilidade

- ⌨️ **Skip-link** para pular direto ao conteúdo principal
- 📱 **Menu drawer mobile** com overlay e controle de `aria-expanded`
- 🖱️ **Scroll suave** entre seções via JavaScript
- 🎯 **Header fixo** com `backdrop-filter` e detecção de rolagem
- 🔤 **Tipografia responsiva** com `clamp()` para o título principal
- ♿ **Textos ocultos visualmente** (`.visually-hidden`) para leitores de tela
- 🖼️ Cards de projeto com layouts adaptáveis (web e mobile)

## ⚙️ Como Rodar Localmente

```bash
# 1. Clone o repositório
git clone https://github.com/SamilisBrito/portfolio.git
cd portfolio

# 2. Instale as dependências
npm install

# 3. Compile o TypeScript
npm run build

# 4. Abra o index.html no navegador
# Ou use uma extensão como Live Server no VS Code
```

### Scripts Disponíveis

| Comando | Descrição |
|---|---|
| `npm run build` | Compila o TypeScript para JavaScript (`dist/`) |
| `npm run watch` | Modo watch — recompila automaticamente ao salvar |

## 🌐 Deploy

O projeto utiliza **GitHub Pages** com deploy automático via **GitHub Actions**. A cada push na branch `main`, o workflow `static.yml` faz o deploy do conteúdo estático.
